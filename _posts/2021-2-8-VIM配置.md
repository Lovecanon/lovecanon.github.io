
# VIM配置


### 升级vim
```bash
# 删除vim
yum list installed | grep vim
yum remove -y vim-common

# 安装依赖
yum install -y gcc make ncurses ncurses-devel
yum install -y ctags tcl-devel python36-devel

# 安装vim。使用python3interp解决py3file报错
git clone https://github.com/vim/vim.git
cd vim
./configure --with-features=huge --enable-multibyte --enable-python3interp --enable-gui=no

make
make install

# 重开bash，查看vim版本
vim --version | head -1


```

### vim-clangd安装
```bash
# 安装依赖
yum -y install glibc
# yum直接安装cmake会报错，通过官网安装脚本安装。
wget cmake-3.21.0-linux-x86_64.sh
chmod +x cmake-3.21.0-linux-x86_64.sh
./cmake-3.21.0-linux-x86_64.sh


git clone --depth=1 https://github.com/llvm/llvm-project.git
cd llvm-project
mkdir build && cd build
cmake -G "Unix Makefiles" -DLLVM_ENABLE_PROJECTS="clang;clang-tools-extra" -DCMAKE_INSTALL_PREFIX=~/tools/llvm -DCMAKE_BUILD_TYPE=Release ../llvm
make -j 16
make install

```
参考：

* [clangd安装教程](https://jdhao.github.io/2021/07/03/install_clangd_on_linux/)
* [cmake官网](https://cmake.org/download/)

```bash
# 添加：export EDITOR=/usr/bin/vim
$ vim /root/.bashrc
```
